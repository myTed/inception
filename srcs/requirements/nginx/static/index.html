웹 페이지란

웹 페이지는 웹 브라우저에서 보는 각각의 화면을 말한다. 웹 페이지는 기본적으로 HTML과 CSS 로 구성된 텍스트 파일 및 관련된 데이터 파일로 구성된다.

웹 서버는 클라이언트 측에 있는 웹브라우저의 요청에 따라 해당 웹페이지의 HTML/CSS 파일과 관련 데이터 파일(이미지, 동영상, 음성)을 클라이언트에 전송하고, 클라이언트 측의 웹 브라우저는 서버로 부터 받은 데이터 파일을 해석하여 화면에 보여준다.

웹 이란

  • 웹은 거미줄을 의미하는데 WWW(World Wide Web)의 약어
  • 인터넷과 웹 브라우저를 통하여 정보를 사용자에게 제공
  • 웹브라우저를 통하여 원격에 있는 사용자와 상호간에 소통
  • 웹 관련 직업: 웹 디자이너, 웹 퍼블리셔, 웹 프로그래머
